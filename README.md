# CosUI
Nice lil ui for cosmic. Color based with multiple themes including blue, green, purple, red, obsidian, dark and light. DM emerald_oree. on discord for suggestions/issues
Yes I did use AI for a good portion of this, but everythings been tested and designed manually.

## Requirements
- [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) (already installed on most Windows 11 machines)
